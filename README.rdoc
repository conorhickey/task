== README
* Ruby version
1.9.2

Application:
Application is a gear log in this case for a Kayak club. 
The Admin (Equipments Officer) can add gear and students to the site, they can also create more admins.
Users viewing the site can see all the gear in the club and further details on this gear.
They can also view each student and what gear each student owns.
On each students gear page it will show if a gear is borrowable or sellable and if it is sellable the price of the gear.
